# Sistema de Biblioteca em Java

Sistema de gerenciamento de biblioteca desenvolvido em Java com SQLite.

## Funcionalidades

-  Login de usuários (Admin/Leitor)
-  Cadastro de livros e usuários
-  Empréstimo e devolução de livros
-  Cálculo automático de multas (R$ 0,35/dia)
-  Interface por linha de comando

## Tecnologias

- Java
- SQLite
- JDBC

## Como executar

1. Importe o projeto no Eclipse
2. Execute `SistemaBiblioteca.java`
3. Use admin/admin para login administrativo
